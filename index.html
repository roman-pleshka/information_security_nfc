SICHERHEITSWARNUNG // NFC
📡
⚠ WARNUNG

DU HAST EINEN NFC-TAG GESCANNT

// SO BEGINNEN DIE MEISTEN ANGRIFFE

Du hast gerade automatisch einen Link geöffnet — ohne nachzudenken.

Einem echten Angreifer würde das reichen, um Schadsoftware zu installieren oder deine Daten zu stehlen.

// Was hätte passieren können:

🎣

Phishing

Eine gefälschte Login-Seite — du gibst dein Passwort ein und übergibst es direkt dem Angreifer.

💀

Drive-by Download

Das Öffnen einer URL kann automatisch einen Datei-Download auf dein Gerät auslösen.

🔗

Weiterleitungskette

Der Tag führt zu einer „sicheren" Seite, die dich dann auf eine schädliche weiterleitet.

✓ So schützt du dich:

01

URL prüfen bevor du öffnest — stimmt sie mit der erwarteten Quelle überein?

02

NFC deaktivieren wenn du es nicht benötigst — reduziere deine Angriffsfläche.

03

Keine unbekannten Tags scannen — besonders an öffentlichen Orten.

04

NFC-Reader-App nutzen um den Tag-Inhalt vorab zu prüfen, bevor dein Gerät handelt.
